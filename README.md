# Aura support

Public place to **report bugs**, **request features**, and read the **privacy policy** for **Aura** — a Mac app that turns speech into useful work.

This repository contains **no application source code**. The Aura app itself is not open source.

## Privacy policy

**[Privacy Policy](PRIVACY.md)** — what Aura captures, what stays on your Mac, and when data can leave your device.

Published URL for App Store Connect:

`https://github.com/dipockdas/aura-support/blob/main/PRIVACY.md`

## Report an issue

Use [**GitHub Issues**](https://github.com/dipockdas/aura-support/issues/new/choose) and pick a template:

- **Bug report** — something broken or unexpected
- **Feature request** — an idea for improvement

Please include your macOS version and Aura version (**Aura → About Aura**) when reporting bugs.

## Private feedback

For sensitive matters, open a [GitHub Issue](https://github.com/dipockdas/aura-support/issues/new/choose) and note that you would like a private follow-up. Do not post secrets, API keys, or personal transcript content in public issues.
